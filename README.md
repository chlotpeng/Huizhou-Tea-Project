# 徽州茶事 · 节气寻茶

这是一个基于 Python 和 Pygame 开发的交互式茶文化体验小程序。

## 项目简介
项目旨在通过“节气寻茶”与“茶田种植”两个板块，带领用户深入了解安徽名茶（如黄山毛峰、六安瓜片等）及其背后的文化内涵。

## 特色功能
- **节气图鉴**：沉浸式查看五个节气的茶山风景与文化信息。
- **手势交互**：集成 MediaPipe 手势识别，实现非接触式浏览与操作。
- **茶田模拟**：包含茶叶种植、生长、采摘的游戏化交互逻辑。
